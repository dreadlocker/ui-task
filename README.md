# ui-task
## How to start the project
* open terminal
* git clone https://github.com/dreadlocker/ui-task.git
* cd ui-task
* npm i
* npm run serve
* if you see this errors (Expected linebreaks to be 'LF' but found 'CRLF') - rerun npm run serve
* in the browser open http://localhost:8080/
